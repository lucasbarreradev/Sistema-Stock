# 🖥 Sistema de Control de Stock (Desktop)

## 📑 Descripción
Aplicación de escritorio desarrollada en VB.NET para la gestión de stock de un negocio.

El sistema permite administrar productos, clientes, proveedores y ventas, incluyendo control de cuentas corrientes y generación de reportes.

## ⚙️ Funcionalidades
- 📦 Gestión de productos
- 👤 Gestión de clientes y proveedores
- 🛒 Módulo de ventas
- 💳 Control de cuentas corrientes
- 📊 Reportes de ventas con filtros por fecha y cliente
- 🗄 Diseño y gestión de base de datos relacional

## 🛠 Tecnologías
- VB.NET
- Windows Forms
- SQL Server
- ADO.NET

## 📦 Requisitos
- Windows
- .NET Framework
- SQL Server

## ⚙️ Configuración
1. Clonar el repositorio
2. Configurar la cadena de conexión a la base de datos
3. Ejecutar el proyecto desde Visual Studio

## 🚀 Uso
1. Registrar clientes, proveedores y productos
2. Realizar ventas desde el módulo correspondiente
3. Consultar cuentas corrientes
4. Generar reportes de ventas

## 🗄 Base de datos
El sistema utiliza una base de datos relacional en SQL Server diseñada para gestionar:
- Clientes
- Proveedores
- Productos
- Ventas
- Cuentas corrientes

## ⚠️ Notas
- Asegurarse de que SQL Server esté en ejecución
- Verificar la cadena de conexión antes de iniciar
